The changelog of Diacritic Buster

1.1.x.x (#PUTTHERELEASEDATEHERE): 
* the program appears in the system tray — it is also possible to hide it on its startup; 
* Clipboard content convertion possibility using the Alt+C hotkey combination; 
* multi-character (string) keys support — allows to work with digraphs and longer combinations; 
* attached Katakana schemes; 
* removed the bug with repeating keys in scheme files; 
* fixed the "Romanise Russian letters the GOST 2002(B) method" scheme; 
* changed the software licence to MIT.

1.0.5479.35147 (2015-01-01) — first public release version.
